# Party Dispenser HACS Integration

A Home Assistant Community Store (HACS) plugin — Python integration plus custom Lovelace card — for the Party Dispenser backend. Replaces the existing YAML package with a one-click HACS install, UI config flow, proper HA entities, and a purpose-built dashboard card.

**Status:** Phase 1 scaffold. The integration loads as a no-op; Phases 2–6 add config flow, entities, services, realtime push, and the custom card.

## Install (once v1.0 ships)

_Installation instructions land in Phase 6. For now this is a development-only scaffold._

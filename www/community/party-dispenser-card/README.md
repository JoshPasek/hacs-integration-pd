# Party Dispenser Card (Phase 4)

This directory is a Phase 1 placeholder. The Party Dispenser custom Lovelace card (lit-element + TypeScript + rollup) is built in Phase 4.

**Do NOT ship a `.js` file here in Phase 1.** Per 01-RESEARCH.md Pitfall 1, HACS supports exactly one category per repository. Phase 1 publishes as HACS category `integration`; a card shipped from this path before the Phase 4 decision on embedded-card vs separate-repo would be ignored by HACS and cause confusion.

See `.planning/ROADMAP.md` Phase 4 for the build plan.
